## Recommended project sequence

The following sequence turns the checklist into portfolio-grade outputs.

1. **Time-Series From Dyno to Markets**
   - [ ] Compare engineering sensor traces with financial return series.
   - [ ] Study stationarity, autocorrelation, volatility clustering, and heavy tails.
   - [ ] Produce a short report explaining which engineering modelling instincts transfer and which fail.

2. **Return Forecasting Baselines**
   - [ ] Implement naive, moving-average, regression, ridge/lasso, and ARIMA baselines.
   - [ ] Validate using walk-forward splits.
   - [ ] Track transaction costs, turnover, and drawdown.

3. **Portfolio Risk and Optimisation**
   - [ ] Build mean-variance optimisation and efficient frontier tools.
   - [ ] Add Sharpe ratio, beta, alpha, drawdown, VaR, and expected shortfall.
   - [ ] Add PCA and factor-model risk decomposition.

4. **Backtesting Engine**
   - [ ] Build reusable backtesting modules for orders, fills, costs, portfolio state, metrics, and validation.
   - [ ] Add tests for cash accounting, position accounting, slippage, transaction costs, and no look-ahead bias.

5. **Options Pricing and Greeks**
   - [ ] Implement Black-Scholes, Greeks, put-call parity, Monte Carlo pricing, and implied volatility inversion.
   - [ ] Add delta hedging simulation and model-risk commentary.

6. **Gaussian Process / Volatility Research**
   - [ ] Apply Gaussian process regression to volatility forecasting or volatility surface interpolation.
   - [ ] Compare SE, RQ, Matérn, ARD, and isotropic kernels.
   - [ ] Report uncertainty bands and out-of-sample validation.

7. **Market Microstructure and Execution**
   - [ ] Study bid-ask spread, liquidity, order-book imbalance, slippage, market impact, and execution cost modelling.
   - [ ] Build an execution-aware strategy evaluation framework.

8. **Advanced Strategy Research**
   - [ ] Explore cointegration, statistical arbitrage, regime switching, reinforcement learning for factor portfolios, and orthogonal risk-integrated alpha.
